Note: This test was developed using following tools and frameworks, so it assumes you have the required plugins for testNG and Maven in your IDE to run the test.

Eclipse - IDE
Maven - Dependency management
TestNG - Test framework

To run the test:

1. Unzip the file and extract the folder.
2. Import the project to your IDE locating the folder.
3. Locate the test at src/main/java -> com.chewy.test.seleniumTest.chewy.tests -> ChewySampleTest
4. Right-click the test file and run the test as TestNG
